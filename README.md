🛒 GreenCART
- A production-style full-stack grocery e-commerce platform built as an advanced monolith, focusing on correctness, performance, security, and real-world business workflows.

🚀 Overview
GreenCART is designed to simulate a real production backend system rather than a basic CRUD demo.
The project emphasizes:
- Secure authentication & authorization
- Order lifecycle management
- Inventory consistency
- Payment integration
- Role-based access control
- API protection strategies
- Clean layered backend architecture

🔐 Security Features
- JWT-based authentication
- Role-based authorization (User / Admin)
- Protected admin APIs
- Environment-based secret management
- Secure payment handling with Stripe
- Google OAuth authentication

🛍 Core Features

👤 User
- Register / Login (JWT + OAuth)
- Browse products
- Add to cart
- Checkout & place orders
- Payment integration
- Order history tracking

🛠 Admin
- Product management (CRUD)
- Inventory updates
- Order status management
- Protected admin dashboard APIs
